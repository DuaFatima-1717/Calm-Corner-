                                             Calm Corner 🌸
Role: Frontend Developer – Diet Space Module

Calm Corner is a wellness-focused web app built using HTML, CSS, and JavaScript as part of the Kode With Klossy program. The app helps users cultivate healthy routines and track their wellbeing through five core features:

Mood Tracker: Log daily moods and track emotional patterns

Journal: Reflect on thoughts and experiences

Diet Space: Plan meals, log nutrition, and explore healthy eating habits

Medication: Track medications and set reminders

Exercise: Log workouts and monitor physical activity

My Contribution: Designed and implemented the Diet Space module, creating intuitive pages for meal planning and nutritional tracking.

Collaborated with a 5-member team to build a cohesive, Studio Ghibli–inspired user interface and seamless user experience.
Focused on making the diet pages interactive, visually appealing, and user-friendly, integrating input forms, logging features, and clear nutritional displays.

Tech Stack: 

● Frontend: HTML, CSS, JavaScript

● Design: UI/UX inspired by Studio Ghibli aesthetics

Collaboration: Worked closely with team members to ensure consistent style and functionality

Calm Corner combines functionality and design to create a calm, motivating space for wellness tracking, helping users build healthier routines with ease and enjoyment.
